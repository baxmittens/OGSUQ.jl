OpenGeoSysUncertaintyQuantification.StochasticOGS6Parameter ‚Äî Type
mutable struct StochasticOGS6Parameter

Container for the definition of a stochastic OGS6 parameter. For all distributions a lower and upper bound have to be provided. For Distributions.Uniform, this can be the interval $[a,b]$, for Distributions.Normal, proper bounds have to be provided.

Fields

  • path::String : OGS6 path definition (see Ogs6InputfileHandler.getAllPathesbyTag and Ogs6InputFileHandler.getElementbyPath).
  • valspec::Int : Value specifier (1 for scalar parameters, $i$ for $i$-th value of a tensor parameters written as a vector in OGS6 e.g. [$a_1=a_{11}, a_2=a_{12}, a_3=a_{21}, a_4=a_{22}$]).
  • dist::UnivariateDistribution : Univariate distribution (see Distributions.UnivariateDistribution).
  • lower_bound::Float64 : Lower bound for truncated distribution (see Distributions.truncated).
  • upper_bound::Float64 : Upper bound for truncated distribution (see Distributions.truncated).

OpenGeoSysUncertaintyQuantification.SparseGridParams ‚Äî Type
mutable struct SparseGridParams <: SampleMethodParams

Container for sparse grid paramters. For more information on the adaptive sparse grid, see the documentation.

Fields

  • N::Int : Dimension of the sparse grid, e.g. the stochastic state space.
  • CT::Type : Type of the collocation points, normally Float64.
  • RT::Type : Return type of the function fun calling the OGS6 binary. Can be customized to Float64, Vector{Float64}, Matrix{Float64}, XDMFFileHandler.XDMF3File, VTUFileHandler.VTUFile, or a custom data type which have to implement the AltInplaceOpsInterface.
  • pointprobs::Vector{Int} : Point set properties of the sparse grid, see DistributedSparseGrids.init and the point set example.
  • init_lvl::Int : Number of initial hierarchical levels before adaptive refinement starts.
  • maxlvl::Int : Number of additional hierarchical level after init_lvl.
  • tol::Float64 : tolerance for adaptive refinement.
  • file::String : path to file to write the StochasticOGSModelParams as XML-file by XMLParser.Julia2XML.

mutable struct OGSUQParams

Container for stochastic model parameters and sample method parameters.

Fields

  • stochasticmodelparams::StochasticOGSModelParams : stochastic model parameters.
  • samplemethodparams::SampleMethodParams : sample method parameters:

Can be instantiated by 

OGSUQParams(
	stochasticmodelparams::StochasticOGSModelParams, 
	samplemethodparams::SampleMethodParams
	)

or 

OGSUQParams(
	file_stochasticmodelparams::String, 
	file_samplemethodparams::String
	)

where file_stochasticmodelparams and file_samplemethodparams are pathes to xml files of StochasticOGSModelParams and SampleMethodParams, respectively, written by XMLParser.Julia2XML.

Helper function for initial setup of a stochastic project. Scans an existing OGS6 projectfile (.prj) for keywords indicating numeric values usable as stochastic input parameter. Returns a Vector of Strings with OGS6 pathes. Uses OGS6 Pathes for parameter indentification (see Ogs6InputfileHandler.getAllPathesbyTag and Ogs6InputFileHandler.getElementbyPath).

Arguments

  • projectfile::String: Path to OGS6 project file (e.g. ./path/to/project.prj)
  • file::String: Path of output XML file which gets written by this function.
  • keywords::Vector{String}: List of keywords.

OpenGeoSysUncertaintyQuantification.dependend_tensor_parameter! ‚Äî Method
dependend_tensor_parameter!(modeldef::Ogs6ModelDef, stoparam::String, master_ind::Int, slave_ind::Int, depfunc)

Helper function to model a dependency of entries of a tensor parmeter such as the permeability tensor k. Uses OGS6 Pathes for parameter indentification (see Ogs6InputfileHandler.getAllPathesbyTag and Ogs6InputFileHandler.getElementbyPath).

Arguments

  • modeldef::Ogs6ModelDef: OSG6 Model Definition
  • stoparam::String: OGS6 Path to parameter
  • master_ind::Int: master index
  • slave_ind::Int: slave index
  • depfunc: dependency function

Example:

dependend_tensor_parameter!(
	modeldef, 
	path_to_permeability, 
	1,
	3, 
	x->0.5*x)

The above example sets the third entry of the permeability tensor to half of the value of the first entry.

Helper function to model a dependency of two OGS6 parameter. Uses OGS6 Pathes for parameter indentification (see Ogs6InputfileHandler.getAllPathesbyTag and Ogs6InputFileHandler.getElementbyPath).

Arguments

  • modeldef::Ogs6ModelDef: OSG6 Model Definition
  • masterstoparam::String: OGS6 Path to master parameter
  • slavestoparam::String: OGS6 Path to slave parameter
  • master_ind::Int: master index
  • slave_ind::Int: slave index
  • depfunc: dependency function

Example:

dependend_tensor_parameter!(
	modeldef, 
	path_to_permeability_id1, 
	path_to_permeability_id2, 
	1, 
	1, 
	x->x)

The above example couples two permeabilites of different material layers.
